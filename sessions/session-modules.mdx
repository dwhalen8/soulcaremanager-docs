The Session View is built around modules — individual panels you can turn on or off depending on how you structure your sessions. This lets you build a Session View that matches your actual counseling workflow rather than forcing you into a fixed format.

## Available modules

| Module | What it does |
|---|---|
| **Presenting Focus** | A short field to note the presenting concern or focus for this specific session |
| **Session Notes** | Your main rich-text note area — the primary documentation space |
| **Weekly Check-In** | A 1–10 rating plus a short notes field for how the counselee's week went |
| **Homework Review** | Shows homework assigned from the previous session; lets you note what was completed |
| **Assigned Lesson** | Displays a lesson resource you've assigned, so you can work through it together |
| **PDI Preview** | A read-only view of the counselee's submitted intake form, visible during the session |
| **File Attachments** | Attach files directly to this session |
| **Session History** | A sidebar panel showing the counselee's prior sessions and notes at a glance |

## Turning modules on and off

Modules are configured through your session template. To adjust which modules appear:

<Steps>
  <Step title="Open Session Templates">
    Go to **Sessions** in the sidebar and find the **Templates** section.
  </Step>
  <Step title="Edit your template">
    Open the template you use and toggle modules on or off.
  </Step>
  <Step title="Save">
    The updated module layout applies the next time you open a session using that template.
  </Step>
</Steps>

See [Session Templates](/sessions/session-templates) for more on creating and managing templates.

## Quick-create homework

From within the Session View, you can create a new homework resource on the spot without leaving the page. Look for the **Add Homework** shortcut in the homework area. The resource is created in your library and assigned to the counselee immediately.

## Carry-over homework

The homework review module surfaces any homework from previous sessions that is still open — not yet marked complete. This lets you quickly carry items forward to the current week's plan with one click, rather than re-assigning them manually.

## Rich text formatting

The Session Notes field supports full rich-text formatting: headings, lists, bold, italic, underline, links, and inline images. Use formatting to organize your notes in a way that's readable later.

## Scripture references

You can add Scripture references to a session note, making it easy to search your notes later by passage.
