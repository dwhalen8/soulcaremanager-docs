---
title: "Session Templates"
description: "Save and reuse module configurations for different kinds of sessions."
---

Session templates let you save a particular module configuration so you don't have to set it up each time. If you run different kinds of sessions — initial consultations, regular follow-ups, joint sessions — you can have a template for each.

## Types of templates

| Type | Who can see it |
|---|---|
| **Personal template** | Only you |
| **Center template** | All counselors at your center |

Center templates are created by center admins and provide a consistent baseline for the whole practice. You can create personal templates on top of those for your own preferred variations.

## Create a template

<Steps>
  <Step title="Go to Session Templates">
    Click **Sessions** in the sidebar, then find the **Templates** section.
  </Step>
  <Step title="Click New Template">
    Give it a name that describes the session type it's for.
  </Step>
  <Step title="Configure the modules">
    Toggle modules on or off and arrange them in the order you want.
  </Step>
  <Step title="Set the scope">
    Choose whether this is a personal template or a center-level template (admins only).
  </Step>
  <Step title="Save">
    The template is now available when starting a session.
  </Step>
</Steps>

## Apply a template to a session

When you open a session, you can select which template to use from a dropdown at the top of the Session View. Switching templates updates the visible modules immediately.

## Clone a template

If you want a variation of an existing template — for example, a center template you want to personalize — you can clone it into your own personal scope.

Open the template, use the **Clone** option, and the copy is saved as a personal template you can edit freely without affecting the original.

## Default template

You can set one template as your default so it's pre-selected every time you open a new session. Go to your templates list and mark the one you use most often as default.
