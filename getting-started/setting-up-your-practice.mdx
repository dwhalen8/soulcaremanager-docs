<Note>
  This page is for center administrators. If you're a counselor joining an existing center, your admin will handle these steps.
</Note>

## 1. Add your branding

Go to **Settings → Center Settings** to upload your logo, set your primary color, and add your practice name and tagline. Your branding appears on emails sent to counselees, exported PDFs, and the counselee portal.

You can upload two logo variants — one for light backgrounds and one for dark — so the right one appears in the right context.

## 2. Set your scheduling configuration

Under **Settings → Center Settings**, configure how appointments are booked:

- **Working hours** — which days and hours counselors are generally available
- **Default appointment length**
- **Buffer between sessions** — a gap automatically added between appointments
- **Booking lead time** — how far in advance a counselee can book
- **Booking window** — how far out into the future they can schedule

These settings drive the public self-scheduling page where prospective counselees can request appointments.

## 3. Configure intake

Decide whether new counselees should automatically receive a Personal Data Inventory (PDI) when they're added to the system. Toggle **Auto-send PDI** on or off under Center Settings.

The PDI is a four-page biblical counseling intake form included out of the box. You can customize it or build your own — see [Forms & Intake](/forms/the-default-pdi) for details.

## 4. Invite your counselors

New users join through invitation codes. To invite a counselor:

<Steps>
  <Step title="Go to the admin panel">
    Navigate to **Settings → User Management** or the admin area.
  </Step>
  <Step title="Generate an invitation code">
    Create a code and set its intended role (counselor, intake staff, etc.) and expiry.
  </Step>
  <Step title="Share the code">
    Send the code to the person you're inviting. They'll use it when creating their account at the registration page.
  </Step>
</Steps>

<Tip>
  Invitation codes can be set to single-use or limited multi-use, so you can send one code to a group or a unique code to each person.
</Tip>

## 5. Set up billing

If you haven't already subscribed, go to **Settings → Billing** to start your subscription. Soul Care Manager is billed per active counselor seat on a monthly or annual basis. See [Billing & Subscription](/billing/how-billing-works) for full details.

## You're ready

Once branding, scheduling, and at least one counselor are in place, your practice is ready to use. From here, counselors can start adding counselees, scheduling sessions, and sending intake forms.
