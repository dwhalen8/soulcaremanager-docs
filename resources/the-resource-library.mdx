title: "The Resource Library"
description: "A central library of articles, lessons, and homework you can assign to counselees."
---

The resource library is where you store and manage all the teaching content you use in counseling — articles, homework assignments, lessons, and devotionals. From the library you can assign resources to counselees, build lessons with bundled homework, and access a curated catalog of premade content.

## Resource categories

| Category | Use case |
|---|---|
| **Homework** | Assignments for counselees to work through between sessions — includes devotionals, Scripture studies, and written exercises |
| **Articles** | Reading material you want a counselee to engage with |
| **Lessons** | Structured teaching content you work through together in session; can include default homework that assigns automatically |

## The premade catalog

Soul Care Manager includes a platform-curated catalog of biblical counseling resources available to every center. These cover common counseling topics and are a good starting point if you're building your library from scratch.

To use a premade resource:

<Steps>
  <Step title="Go to Resources">
    Click **Resources** in the sidebar.
  </Step>
  <Step title="Browse the premade catalog">
    Find the **Premade** or **Catalog** tab.
  </Step>
  <Step title="Adopt a resource">
    Click **Add to My Library** on any resource. A copy is created in your library that you can edit freely — changes to your copy don't affect the catalog original.
  </Step>
</Steps>

## Creating your own resources

<Steps>
  <Step title="Go to Resources">
    Click **Resources** in the sidebar.
  </Step>
  <Step title="Click New Resource">
    Choose a category and give the resource a title.
  </Step>
  <Step title="Add content">
    Write the resource content using the rich-text editor. You can also attach files — PDFs, Word documents, images, or anything else relevant.
  </Step>
  <Step title="Add tags and Scripture references">
    Tags and Scripture references make resources easier to find later.
  </Step>
  <Step title="Save or publish">
    Save as a draft if you're not finished, or publish when it's ready to assign.
  </Step>
</Steps>

## Duplicating a resource

To create a variation of an existing resource — a shorter version, or one adapted for a specific counselee — use the **Duplicate** button on any resource you own. The copy is independent of the original and can be edited freely.

## Bulk import

If you have existing materials in Word documents or a spreadsheet, you can import them into your library in bulk rather than creating each resource manually. Go to **Resources** and look for the **Import** option. Supported formats are CSV and .docx.

## Promoting a resource to the center library

Center admins can promote any personal resource to the center library, making it available to all counselors. Open the resource and use the **Copy to Library** option.
