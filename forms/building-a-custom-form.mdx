---
title: "Building a Custom Form"
description: "Use the drag-and-drop form builder to create intake forms, surveys, and feedback forms."
---

Beyond the default PDI, you can build your own forms for any purpose — a pre-session survey, a feedback form, a supplemental intake questionnaire, or anything else your practice needs.

## Create a new form

<Steps>
  <Step title="Go to Forms">
    Click **Forms** in the sidebar.
  </Step>
  <Step title="Click New Form">
    Give the form a name and choose a category.
  </Step>
  <Step title="Build your pages and fields">
    Use the drag-and-drop builder to add fields.
  </Step>
  <Step title="Save">
    The form is saved as a draft until you're ready to send it.
  </Step>
</Steps>

## Form categories

| Category | Use case |
|---|---|
| **PDI** | Primary intake form; only one can be set as default per center |
| **Intake / Contact** | Supplemental intake or initial contact forms |
| **Survey** | Mid-counseling or ongoing check-ins |
| **Feedback** | End-of-counseling feedback |
| **General** | Anything else |

## Field types

The form builder supports the following field types:

- **Short text** — single-line input
- **Long text** — multi-line text area
- **Email** — validated email address field
- **Phone** — phone number input
- **Date** — date picker
- **Dropdown** — select one from a list
- **Radio buttons** — select one from visible options
- **Checkboxes** — select one or more
- **File upload** — let the counselee attach a file
- **Signature** — drawn or typed signature
- **Text / HTML block** — instructions, consent language, section headers, or any static content

## Multi-page forms

Forms can span multiple pages. Add a new page using the page controls in the builder. Each page is presented to the counselee one at a time, with a progress indicator showing how far along they are.

## Mapping fields to the counselee profile

For intake forms, you can map specific fields to counselee profile fields. When the form is submitted, the mapped data automatically populates the counselee's record.

To map a field, open its settings in the builder and select the corresponding profile field from the dropdown.

## Required fields

Mark any field as required to prevent submission without an answer. Required fields are indicated with an asterisk when the counselee fills out the form.

## Consent and signature fields

Use an HTML/text block to add consent language, then follow it with a signature field. The signature is captured and stored with the submission.

## Making a form the default PDI

If you've built a custom intake form you want to use as your primary PDI, you can promote it to the default. Go to the form's settings and select **Make Default**. Only one form per category can be the default at a time.
